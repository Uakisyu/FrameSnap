# FrameSnap
Bat Script, uses ffmpeg to extract frames from a video at a specified rate and save them in a folder.  

Step 1 (Download ffmpeg):  
(FFmpeg builds from gyan.dev)[https://www.gyan.dev/ffmpeg/builds/]  
After downloading, place it in the directory where the .bat script is located.

Step 2 (Input settings):  
Follow the prompts to enter the information required by the script. The video information should be the video name (the video needs to be placed in the directory where the script is located) or the complete video path.  

Step 3 (Run / End):  
After you have entered all the information, press anykey, and ffmpeg should start and work.
